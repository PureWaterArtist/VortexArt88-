# 📐 VortexArt88 CAD Models

This folder is dedicated to the parametric geometries, nozzle models, and chamber housings for the project.

### 🤝 Open Collaboration
If you have modeled the **Figure-8 Collision Chamber** based on the specifications in the main README, please submit a **Pull Request** uploading your `.step`, `.f3d`, or `.stl` files directly into this directory.
