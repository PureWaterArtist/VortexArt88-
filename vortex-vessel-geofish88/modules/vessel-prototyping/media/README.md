## 📸 Technical Asset Verification & Field Capture Protocols

To maintain absolute data integrity and verifiable prior-art across all open-source submissions to the public ledger, all workshop photographs, diagnostic tracking charts, or tank-test video snapshots uploaded here by the community must strictly adhere to these baseline formatting rules:

1.  **High-Contrast Production Renders:** All CAD layout or machine-queue software vector exports must use a clean, dark-mode background profile to ensure that spindle tracking paths and nesting margins are perfectly scannable on workshop monitors.
2.  **CNC Carving Verification Documentation:** High-resolution macro photography demonstrating successful 2.2 mm V-groove scores must be recorded under backlit cross-lighting to visually confirm a consistent remaining 1.8 mm live-hinge web.
3.  **Hydrostatic Displacement Logs:** Visual data overlays tracking the waterline draft depth inside a testing bay must match the precise mathematical calculations verified inside the root `geofish88-geometry-twin.py` digital twin script.
