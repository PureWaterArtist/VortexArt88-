## 📸 Workbench Telemetry & Visual Capture Protocols

To maintain absolute data integrity and verifiable prior-art across all open-source submissions to the public ledger, all assembly snapshots, line traces, or force testing records uploaded here by the community must strictly adhere to these baseline formatting rules:

1.  **High-Contrast Latch Renders:** All CAD or mechanism layout software vector exports must use a clean, dark-mode background profile to ensure that mechanical link distances and over-center travel limits are perfectly scannable on workshop monitors.
2.  **Gasket Sealing Documentation:** High-resolution macro photography demonstrating successful 35% EPDM bulb compression checks must be shot under sharp cross-lighting to visually confirm a complete hermetic interface.
3.  **Tension Distribution Logs:** Visual data overlays tracking the 450.0 Newton drawing load across individual latches must link directly back to the validation parameters verified inside the root `geofish88-geometry-twin.py` engine.
