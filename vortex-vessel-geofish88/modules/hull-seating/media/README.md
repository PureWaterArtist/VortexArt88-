## 📸 Workbench Telemetry & Visual Capture Protocols

To maintain absolute data integrity and verifiable prior-art across all open-source submissions to the public ledger, all manufacturing snapshots, line traces, or load-bearing records uploaded here by the community must strictly adhere to these baseline formatting rules:

1.  **High-Contrast Internal Renders:** All CAD or structural layout software vector exports must use a clean, dark-mode background profile to ensure that interlocking rail margins and tab clearances are perfectly scannable on workshop monitors.
2.  **Structural Load Documentation:** High-resolution photography demonstrating successful vertical seat stress checks under full target weights must be shot with macro lighting to visually verify zero composite bowing.
3.  **T-Slot Torque Alignment Logs:** Visual data overlays tracking the 2.5 N-m T-bolt tightening forces must match the precise engineering tolerances verified inside the root `geofish88-geometry-twin.py` engine.
