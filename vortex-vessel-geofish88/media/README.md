## 📸 Technical Asset Verification & Cleanroom Camera Protocols

To maintain absolute transparency and ensure that any user can accurately build your buddy's design, any engineering snapshot or rendering uploaded to this public repository must strictly adhere to these layout rules:

1.  **High-Contrast Line Drawings:** Schematic files must use a clean, dark-mode color scheme (such as slate gray lines on near-black backgrounds) to ensure all scores, live-hinge creases, and dimension parameters are easily scannable on workshop terminal screens.
2.  **Uncompressed Format Integrity:** All visual proofs must be committed using uncompressed alpha-channel asset states (`.png` or vector `.svg`) to prevent compression artifacting from obscuring micro-gasket measurements.
3.  **Hydrostatic Data Overlay:** In-water validation photographs tracking actual displacement must include an overlaid text grid noting water density, draft depth in centimeters, and total applied test weight.
