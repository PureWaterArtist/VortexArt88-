## 📸 Technical Asset Verification & Cleanroom Camera Protocols

To maintain absolute data integrity and verifiable prior-art across all open-source submissions to the public ledger, all laser-ablation snapshots, macro material line traces, or vacuum testing records uploaded here by the community must strictly adhere to these formatting rules:

1.  **High-Contrast Production Renders:** All CAD layout or machine-queue software vector exports must use a clean, dark-mode background profile to ensure that laser tracking paths and nesting margins are perfectly scannable on workshop monitors.
2.  **SLA Channel Layer Documentation:** High-resolution macro photography demonstrating successful 0.025 mm layer resolution and uncured resin evacuation flushes must be shot under sharp side cross-lighting to visually confirm zero channel blockage.
3.  **Vacuum Decay Staging Logs:** Visual data overlays tracking the -20.0 kPa joint seal check loop and the strict 5-minute decay threshold must match the precise engineering tolerances verified inside the root `master-grid-twin.py` digital twin engine.
