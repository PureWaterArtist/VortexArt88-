2.  **Input B Only [0, 1]:** If only logic stream B is active, it mirrors this exact same low-friction bypass path, yielding an identical dead state (Output = 0).

Because the fluid columns possess equal and opposing angular momentum vectors, their linear kinetic energy is instantly compressed at the intersection point, triggering an explosive, localized micro-pressure spike.
